# superstore-festival-sales-analysis

## 📌 Overview
This project analyzes Superstore sales data during festival seasons to identify revenue trends, top-performing categories, regional performance, and profit drivers.

## 🎯 Objectives
- Analyze sales performance during festive periods
- Identify high-revenue products and regions
- Compare profit margins across categories
- Provide data-driven recommendations

## 🛠 Tools & Technologies
- SQL (Data extraction & transformation)
- Excel (Data cleaning & preprocessing)
- Power BI (Dashboard & visualization)

## 📊 Key Insights
- Festive sales increased by XX% compared to regular periods
- Technology category generated highest profit
- South region had highest revenue contribution
- Discount levels significantly impacted margins
